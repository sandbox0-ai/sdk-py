# sdk-py examples

Run from `sdk-py`:

```bash
python3 -m pip install -e .
python3 examples/01_hello_world/main.py
```

Environment variables:

- `SANDBOX0_TOKEN` (required)
- `SANDBOX0_BASE_URL` (optional, default `https://api.sandbox0.ai`)
- `SANDBOX0_WEBHOOK_URL` / `SANDBOX0_WEBHOOK_SECRET` (for webhook example)
